3.- Estas son las rutas para cada funcion y ejemplos de insercion:

    1.- Crear un nuevo vehiculo

    http://localhost:3000/vehicle/create/

    {
        "code" : "VH-01",
        "isActive" : true,
        "load" : {
            "material" : "Hierro",
            "weight" : 150
        }
    }

    {
        "code" : "VH-02",
        "isActive" : true,
        "load" : {
            "material" : "Carbon",
            "weight" : 200
        }
    }

    2.- Editar el atributo is active:

    http://localhost:3000/vehicle/update/state/

    {
        "code": "VH-01",
        "state": false 
    }

    3.- Editar la carga:

    http://localhost:3000/vehicle/update/load/

    {
        "code": "VH-01",
        "load": {
            "material" : "crotolamo",
            "weight" : 250
        } 
    }

    4.- Editar la carga:

    http://localhost:3000/vehicle/delete/

    {
        "code": "VH-01"
    }

    4.- Obtener vehiculo por codigo:

    http://localhost:3000/vehicle/find/code/

    Como este es un get se utiliza un parametro y no un body
    key: code value: VH-01

    5.- Obtener todos los vehiculos:

    http://localhost:3000/vehicle/get/all/

    No se ocupa boy ni parametros para esta peticion

4.- ejecuten el archivo "app.js"
